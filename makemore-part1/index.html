01什么是语言模型？

语言模型是给定前文，预测下一个词的概率分布的模型。这是所有 NLP 任务的基础。

数学定义

语言模型学习一个概率分布 P(w1, w2, ..., wn)

输入
"hel"
前 3 个字符
→
语言模型
P(·|hel)
预测下一个
→
输出
"lo"
概率分布

字符级 vs 词级

类型最小单位词汇表大小优点
字符级a-z, 中文字符~100无需处理未知词
词级词~50000语义更强

03MLP 语言模型

Bengio 2003 提出的 MLP 语言模型，将前 block_size 个字符的嵌入拼接后通过 MLP 预测下一个字符。

模型结构

class MLP(nn.Module):
    def __init__(self, config):
        self.wte = nn.Embedding(config.vocab_size + 1, config.n_embd)  # 字符嵌入
        self.mlp = nn.Sequential(
            nn.Linear(config.block_size * config.n_embd, config.n_embd2),
            nn.Tanh(),
            nn.Linear(config.n_embd2, config.vocab_size)
        )

05训练流程

使用交叉熵损失训练语言模型，自动反向传播更新参数。

训练循环

# 1. 前向传播
logits, loss = model(X, Y)

# 2. 计算损失 (交叉熵)
loss = F.cross_entropy(
    logits.view(-1, vocab_size),  # (B*T, V)
    targets.view(-1)               # (B*T,)
)

# 3. 反向传播
loss.backward()

# 4. 更新参数
optimizer.step()
model.zero_grad()

困惑度 (Perplexity)

衡量语言模型质量的指标： 

PP = exp(loss)

# PP 越低，模型越好
# PP = 1: 完美预测
# PP = vocab_size: 随机猜测

06采样生成

使用温度参数控制采样的随机性，自回归生成新名字。

自回归生成

def generate(model, idx, max_new_tokens, temperature=1.0):
    for _ in range(max_new_tokens):
        # 只取最后 block_size 个字符
        idx_cond = idx[:, -model.block_size:]

        # 前向传播
        logits, _ = model(idx_cond)
        logits = logits[:, -1, :] / temperature

        # 采样下一个字符
        probs = F.softmax(logits, dim=-1)
        idx_next = torch.multinomial(probs, num_samples=1)

        # 追加到序列
        idx = torch.cat([idx, idx_next], dim=1)

    return idx

Temperature 控制

Temperature行为效果
1.0原始概率平衡
→ 0贪心选择确定性强，高概率词
→ ∞均匀分布随机性强，创造性高
