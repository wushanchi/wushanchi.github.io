01链式法则

基本公式

# 链式法则
∂L/∂x = ∂L/∂y * ∂y/∂x

# 操作梯度表
加法:  da = dc,  db = dc
乘法:  da = b*dc, db = a*dc
ReLU: da = (a>0)*dc
Softmax: da = p - y

02Cross Entropy 反向传播

梯度公式

# Cross Entropy: CE = -log(p_target)
# 梯度: ∂CE/∂logit_i = p_i - y_i
# 其中 y_i = 1 if i == target else 0

def cross_entropy_grad(logits, target_idx):
    probs = softmax(logits)
    grads = []
    for i, p in enumerate(probs):
        y = 1 if i == target_idx else 0
        grads.append(p - y)
    return grads

03手动反向传播示例

计算图

# 计算: loss = (a * b + c) ** 2
# 其中 a = 2, b = 3, c = 1

# 前向:
ab = a * b        # 6
sum_ab_c = ab + c  # 7
loss = sum_ab_c ** 2  # 49

# 反向:
∂L/∂(a*b+c) = 2 * (a*b+c) = 14
∂L/∂c = 14
∂L/∂a = b * 14 = 42
∂L/∂b = a * 14 = 28
