Input Embedding
Token Representation Layer

Embedding 层将离散的 token ID 映射为连续的 d_model 维向量。每个 token 被转换为一个捕获语义信息的稠密向量。

维度变化: [B, L] → [B, L, D]

Embedding 矩阵形状为 [vocab_size, d_model]。例如词表50000，d_model=512，则参数量为 50000×512 ≈ 2500万。

数据维度
输入→[B, L]
输出→[B, L, 512]

Multi-Head Self-Attention
多头自注意力机制

核心思想: 将注意力分散到多个"头"中，每个头学习不同的注意力模式。

维度变化: 输入 D=512 分成 h=8 个头，每头 d_k=64。

每个头独立计算: Attention(Q_i, K_i, V_i) → 得到8个64维输出

拼接后: [64×8] = 512 → Linear → 最终输出

为什么多头? 不同头可以学习不同类型的依赖关系（语法、语义、指代等）。

维度变化详解
输入X→[B,L,512]
投影Q,K,V→[B,L,512]
分头→[B,h,L,64]
注意力→[B,h,L,L]
拼接→[B,L,512]
输出→[B,L,512]
🎭
Masked Self-Attention
因果遮蔽 / Decoder 自回归

作用: 确保 Decoder 在预测第 t 个 token 时，只能看到位置 1 到 t-1 的信息（不能"作弊"看未来）。

实现方式: 在 softmax 前，将上三角位置设为 -∞

由于 softmax(-∞) = 0，这些位置的注意力权重变为 0，实现因果（causal）特性。

为什么需要?: 训练时如果能看到未来token，模型会"作弊"；推理时未来token还不存在。

维度变化
输入→[B,L,512]
加Mask→[L,L]上三角-∞
输出→[B,L,512]
🔗
Cross Attention
编码器-解码器注意力

核心区别: Q 来自 Decoder，K 和 V 来自 Encoder 输出。

作用: 让 Decoder 的每个位置可以"查询"源序列（Encoder输出）的所有位置，实现跨序列交互。

维度注意: Q 的序列长度是 L_tgt（目标长度），K、V 的序列长度是 L_src（源长度）。

应用: 翻译任务中，Decoder 每生成一个词，可以关注源句子的相关部分。

维度变化
Q→[B,L_tgt,512]
K→[B,L_src,512]
V→[B,L_src,512]
Attn矩阵→[B,L_tgt,L_src]
输出→[B,L_tgt,512]
⚡
Feed Forward Network
前馈神经网络

结构: 两个线性变换，中间有 ReLU 激活: FFN(x) = max(0, xW₁+b₁)W₂+b₂

维度变化: 512 → 2048 → 512

为什么扩展到4倍?: 更大的隐藏层维度让模型有更多容量学习复杂的非线性变换。实验证明4倍是经验最优。

逐位置: FFN 对序列中每个位置的向量独立应用相同变换，是"逐位置"（point-wise）的。

Attention vs FFN: Attention 负责"谁该关注谁"（信息流动），FFN 负责"如何表示"（信息变换）。

维度变化详解
输入→[B,L,512]
W1投影→[B,L,2048]
ReLU→[B,L,2048]
W2投影→[B,L,512]
📊
Layer Normalization
层归一化

公式: LN(x) = γ ⊙ (x - μ) / √(σ² + ε) + β

归一化维度: 在特征维度 D 上归一化，对每个样本、每个位置独立处理。

与 BatchNorm 区别: BatchNorm 在 batch 维度归一化，LayerNorm 在特征维度归一化。LayerNorm 对序列长度变化更鲁棒。

Add & Norm: 残差连接 (x + Sublayer(x)) + LayerNorm。残差连接确保梯度直接回传，避免深层网络梯度消失。

维度变化
输入→[B,L,D]
均值μ→[B,L,1]
方差σ²→[B,L,1]
输出→[B,L,D]
