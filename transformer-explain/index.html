核心模块详解

📥
Input Embedding
Token Representation Layer

Embedding 层将离散的 token ID 映射为连续的 d_model 维向量。每个 token 被转换为一个捕获语义信息的稠密向量。

Embedding 矩阵形状为 [vocab_size, d_model]。例如 GPT-3 的词表约 50,000，d_model=12288。

每个 token 经过 Embedding 后，获得一个固定维度的语义表示向量，捕获了该 token 的语法和语义信息。

数据维度
输入 → [B, L]
输出 → [B, L, D]
关键参数
vocab = ~50k
d_model = 512
🔍
Multi-Head Self-Attention
多头自注意力机制

Self-Attention 是 Transformer 的核心。它允许序列中每个位置关注所有其他位置，捕获长距离依赖。

公式: Attention(Q,K,V) = softmax(QKᵀ/√d_k)V

Multi-Head 将注意力分散到 h 个头中，每个头学习不同的注意力模式（语法、语义、指代等）。

最终输出: MultiHead = Concat(head_1,...,head_h) × W_O

数据维度
输入 → [B, L, D]
Q,K,V → [B, L, D]
注意力 → [B, h, L, L]
输出 → [B, L, D]
参数
h = 8
d_k = 64 (D/h)
🎭
Masked Self-Attention
因果遮蔽 / Decoder 自回归

Masked Self-Attention 确保 Decoder 在预测第 t 个 token 时，只能看到位置 1 到 t-1 的信息。

实现方式：在 softmax 前，将上三角位置设为 -∞。由于 softmax(-∞) = 0，这些位置的注意力权重变为 0。

这是 GPT、LLaMA 等自回归语言模型的核心，确保生成时只能基于已生成的 token。

数据维度
输入 → [B, L, D]
Mask → [L, L]
Mask值 → 上三角=-∞
输出 → [B, L, D]
🔗
Cross Attention
编码器-解码器注意力

Cross Attention 连接 Encoder 和 Decoder，让 Decoder 的每个位置可以关注 Encoder 的所有位置。

关键区别：
• Q 来自 Decoder
• K, V 来自 Encoder

这使得每个 Decoder 位置都能"查询"整个源序列的信息。例如翻译时，Decoder 每个位置可关注源语言的相关部分。

数据维度
Q → [B, L_dec, D]
K, V → [B, L_enc, D]
Attn → [B, L_dec, L_enc]
输出 → [B, L_dec, D]
⚡
Feed Forward Network
前馈神经网络

FFN 是每个 Transformer 层中的两个线性变换，中间有 ReLU 激活：

FFN(x) = max(0, xW₁ + b₁)W₂ + b₂

它是逐位置（point-wise）应用的——对序列中每个位置的向量独立应用相同变换。

维度扩展：D → 4D → D（通常是 d_model 的 4 倍）。Attention 负责"信息流动"，FFN 负责"信息变换"。

数据维度
输入 → [B, L, D]
隐藏 → [B, L, 4D]
输出 → [B, L, D]
参数
d_model = 512
d_ff = 2048 (4×D)
📊
Layer Normalization
层归一化

LayerNorm 对每一层归一化：

LN(x) = γ ⊙ (x - μ) / √(σ² + ε) + β

在特征维度（D）上归一化，对每个样本独立处理。与 BatchNorm 不同，LayerNorm 对序列长度变化更鲁棒。

Add & Norm = 残差连接 (x + Sublayer(x)) + LayerNorm。残差连接确保梯度直接回传。

数据维度
输入 → [B, L, D]
均值 → [B, L, 1]
输出 → [B, L, D]
参数
γ, β = 各 D 个
📍
Positional Encoding
位置编码

Transformer 无循环结构，无法自然捕获位置信息。位置编码通过正弦/余弦函数注入位置信号：

PE(pos, 2i) = sin(pos / 10000^(2i/d))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d))

这种编码方式允许模型学习相对位置关系（因为 sin(A+B) 可表示 A 和 B 的关系）。

现代 LLM 也使用可学习的位置编码或 RoPE（旋转位置编码）等更先进的方法。

数据维度
输入 → [B, L, D]
输出 → [B, L, D]
特点
确定性 → 无需学习
可扩展 → 任意长度

维度符号速查表

B
Batch Size
批大小，同时处理的序列数量，通常 1-64
L
Sequence Length
序列长度，token 数量，通常 128-4096
D
d_model
模型维度，嵌入和隐藏层向量维度，常见 512/768/1024/4096
d_k
Key Dimension
每头 Key 向量维度 = D / h
d_ff
FFN Dimension
FFN 隐藏层维度，通常 4 × d_model
h
Num Heads
注意力头数量，允许并行学习多种注意力模式
V
Vocabulary Size
词表大小，GPT-3: 50257，中文通常 50000+

基于 "Attention Is All You Need" Vaswani et al., NeurIPS 2017
