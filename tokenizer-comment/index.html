01BPE 算法

算法步骤

1. 将文本分解为字节序列
2. 统计相邻字节对频率
3. 合并最频繁的对
4. 重复直到达到词汇表大小

02示例

合并过程

原始: "aaabbc"
字节: [97, 97, 97, 98, 98, 99]

频率: (97,97)=2, (97,98)=1, (98,98)=1, (98,99)=1
最频繁: (97, 97) → 合并为 256

新序列: [256, 97, 98, 98, 99]

03Tokenizer 接口

基本接口

# 编码
tokens = tokenizer.encode("Hello, world!")
# [15496, 11, 1917, 0]

# 解码
text = tokenizer.decode(tokens)
# "Hello, world!"
