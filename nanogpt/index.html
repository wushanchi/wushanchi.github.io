01GPT-2 模型架构

GPT-2 是一个基于 Transformer 的自回归语言模型，包含 Token Embedding、位置编码、多个 Transformer 块和 LM Head。

核心组件

GPT-2 (124M)
├── Token Embedding Table: 50257 × 768
├── Position Embedding Table: 1024 × 768
├── Transformer Blocks (×12)
│   ├── LayerNorm
│   ├── CausalSelfAttention
│   │   ├── QKV Linear: 768 → 2304
│   │   └── Output Linear: 768 → 768
│   ├── LayerNorm
│   └── MLP
│       ├── Expand: 768 → 3072 (GELU)
│       └── Contract: 3072 → 768
├── Final LayerNorm
└── LM Head: 768 → 50257

Causal Self-Attention

确保 t 位置只能看到 0..t 的 token（因果掩码）

class CausalSelfAttention(nn.Module):
    def forward(self, x):
        qkv = self.c_attn(x)           # (B,T,3*n_embd)
        q, k, v = qkv.split(self.n_embd, dim=2)

        # 多头: (B,T,n_embd) → (B,n_head,T,head_size)
        q = q.view(B, T, self.n_head, C/self.n_head).transpose(1, 2)

        # Flash Attention (自动因果掩码)
        y = F.scaled_dot_product_attention(q, k, v, is_causal=True)
        return self.c_proj(y)

学习率调度

warmup
0 → max_lr
→
cosine
max_lr → min_lr
def get_lr(it):
    if it < warmup_steps:
        return max_lr * (it+1) / warmup_steps
    decay_ratio = (it - warmup_steps) / (max_steps - warmup_steps)
    coeff = 0.5 * (1.0 + math.cos(math.pi * decay_ratio))
    return min_lr + coeff * (max_lr - min_lr)

05HellaSwag 评估

HellaSwag 是一个常识推理基准，给定上下文选择最合理的结尾。

评估方法

# 每个样本 4 个候选结尾，计算每个候选的平均损失
def get_most_likely_row(tokens, mask, logits):
    shift_logits = logits[..., :-1, :]
    shift_tokens = tokens[..., 1:]
    shift_losses = F.cross_entropy(flat_shift_logits, flat_shift_tokens, reduction='none')

    # 只看候选结尾部分 (mask == 1)
    shift_mask = (mask[..., 1:]).contiguous()
    masked_losses = shift_losses * shift_mask
    avg_loss = masked_losses.sum(dim=1) / shift_mask.sum(dim=1)

    return avg_loss.argmin().item()
