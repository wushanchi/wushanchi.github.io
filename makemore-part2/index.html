01MLP 原理

MLP（多层感知机）语言模型将前 block_size 个字符的嵌入拼接，通过全连接层预测下一个字符。

核心公式

output = W2 * tanh(W1 * concat(embeddings) + b1) + b2

其中:
  - concat(embeddings): 拼接 block_size 个嵌入向量
  - W1: (n_embd * block_size, n_embd2)
  - W2: (n_embd2, vocab_size)
输入 idx
(B, T)
字符索引
→
wte(idx)
(B, T, n_embd)
嵌入
→
拼接
(B, T, n*k)
n_embd × block_size
→
MLP
(B, T, V)
vocab_size

02MLP 架构

滚动拼接策略

embs = []
for k in range(block_size):
    tok_emb = wte(idx)           # (B, T, n_embd)
    idx = torch.roll(idx, 1, 1)  # 滚动索引
    idx[:, 0] = vocab_size        # 填充 BLANK
    embs.append(tok_emb)

x = torch.cat(embs, -1)          # (B, T, n_embd * block_size)
logits = mlp(x)

03核心代码

MLP 实现

class MLP(nn.Module):
    def __init__(self, config):
        self.wte = nn.Embedding(config.vocab_size + 1, config.n_embd)
        self.mlp = nn.Sequential(
            nn.Linear(self.block_size * config.n_embd, config.n_embd2),
            nn.Tanh(),
            nn.Linear(config.n_embd2, self.vocab_size)
        )

    def forward(self, idx, targets=None):
        embs = []
        for k in range(self.block_size):
            tok_emb = self.wte(idx)
            idx = torch.roll(idx, 1, 1)
            idx[:, 0] = self.vocab_size
            embs.append(tok_emb)

        x = torch.cat(embs, -1)
        logits = self.mlp(x)

        if targets is not None:
            loss = F.cross_entropy(logits.view(-1, vocab_size), targets.view(-1))
            return logits, loss
        return logits, None

04维度变化

操作输入维度输出维度说明
Embedding(B, T)(B, T, n_embd)查表得到嵌入
循环 block_size 次(B, T, n_embd)(B, T, n*n)拼接历史嵌入
Linear + Tanh(B, T, n*n)(B, T, n_embd2)隐藏层变换
Linear(B, T, n2)(B, T, V)输出 logits
