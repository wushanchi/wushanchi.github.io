执行流程详解

📥
0. 数据准备 (Data Preparation)
字符级编码 / 输入输出分割

文本 → 字符ID序列：将字符映射为整数 ID，构建词表。

Train / Val 分割：90% 训练集，10% 验证集。

get_batch()：随机采样 B 个序列，每个长度为 T，输出 x (输入) 和 y (目标, 右移一位)。

# vocab 构建 with open('input.txt') as f: text = f.read() chars = sorted(list(set(text))) vocab_size = len(chars) # 65 # get_batch 输出 def get_batch(split): ix = torch.randint(len(data)-block_size, (batch_size,)) x = torch.stack([data[i:i+block_size] for i in ix]) y = torch.stack([data[i+1:i+block_size+1] for i in ix]) return x, y # x:[B,T] y:[B,T]
↓
🔢
1. Token Embedding Table
nn.Embedding(vocab_size, vocab_size)

核心模型：Bigram 模型仅有一个 lookup table，形状为 [vocab_size, vocab_size] = [65, 65]。

前向传播：输入 token IDs [B, T]，查表得到 logits [B, T, C]，C=vocab_size=65。

生成方式：每次取最后一个时间步的 logits，softmax 后采样下一个 token，自回归生成。

class BigramLanguageModel(nn.Module): def __init__(self, vocab_size): super().__init__() self.token_embedding_table = nn.Embedding( vocab_size, vocab_size # [65, 65] ) def forward(self, idx, targets=None): logits = self.token_embedding_table(idx) # idx: [B, T] → logits: [B, T, C] if targets is not None: B, T, C = logits.shape logits = logits.view(B*T, C) targets = targets.view(B*T) loss = F.cross_entropy(logits, targets) return logits, loss
↓
⚡
2. Forward & Loss 计算
交叉熵损失 / 梯度反向传播

维度变化：[B, T, C] → [B*T, C]，view 操作不复制数据，只改变形状语义。

Cross Entropy：loss = -log(prob_of_correct_token)，衡量模型对正确 token 的预测概率。

训练循环：optimizer.zero_grad() → loss.backward() → optimizer.step()

# 训练循环 for iter in range(max_iters): xb, yb = get_batch('train') logits, loss = model(xb, yb) optimizer.zero_grad(set_to_none=True) loss.backward() optimizer.step() # 维度追踪 # xb: [32, 8] yb: [32, 8] # logits: [32, 8, 65] → [256, 65] # targets: [32, 8] → [256] # loss = cross_entropy([256,65], [256])
↓
🎤
3. 自回归生成 (Generation)
Softmax → 采样 → 自回归

自回归：每步生成一个 token，将输出追加到输入序列，循环直到达到最大长度。

采样：torch.multinomial(probs, num_samples=1) 从概率分布中采样，而不是直接取 argmax。

只取最后一个时间步：logits[:, -1, :] 提取最后一个位置的预测，这是 Transformer 中每个位置预测下一个 token 的简化版。

def generate(self, idx, max_new_tokens): # idx: [B, T] 当前上下文 for _ in range(max_new_tokens): logits, _ = model(idx) # [B, T, C] logits = logits[:, -1, :] # [B, C] 只取最后一个 probs = F.softmax(logits, dim=-1) # [B, C] idx_next = torch.multinomial(probs, num_samples=1) # [B, 1] idx = torch.cat((idx, idx_next), dim=1) # [B, T+1] return idx
维度追踪

完整数据流维度变化

原始文本 → "FAREWELL.." 字符序列，示例
encode(text) → [18, 27, 43, 42, 22, 35, ...] [len] 整数ID列表
data tensor → [len] len ≈ 字符总数 (~1M)
get_batch() → [B, T] = [32, 8] x 和 y 各一个 batch
Embedding(idx) → [B, T, C] = [32, 8, 65] 查表：每个 ID → 65维 one-hot 向量
view(...) → [B*T, C] = [256, 65] 展平用于交叉熵计算
cross_entropy → [] (scalar) 标量损失，反向传播起点
generate() → [1, 1] → [1, N] 从 [B=1, T=1] 开始，自回归扩展到 N tokens

import torch import torch.nn as nn from torch.nn import functional as F # ========== 超参数 ========== batch_size = 32 # 并行处理的序列数 block_size = 8 # 每个序列的最大上下文长度 max_iters = 3000 # 训练迭代次数 eval_interval = 300 learning_rate = 1e-2 device = 'cuda' if torch.cuda.is_available() else 'cpu' eval_iters = 200 torch.manual_seed(1337) # ========== 数据准备 ========== # 下载: wget https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt with open('input.txt', 'r', encoding='utf-8') as f: text = f.read() chars = sorted(list(set(text))) # 所有唯一字符 vocab_size = len(chars) # 65 stoi = { ch:i for i,ch in enumerate(chars) } itos = { i:ch for i,ch in enumerate(chars) } encode = lambda s: [stoi[c] for c in s] decode = lambda l: ''.join([itos[i] for i in l]) data = torch.tensor(encode(text), dtype=torch.long) n = int(0.9 * len(data)) train_data = data[:n] val_data = data[n:] def get_batch(split): data = train_data if split == 'train' else val_data ix = torch.randint(len(data) - block_size, (batch_size,)) x = torch.stack([data[i:i+block_size] for i in ix]) y = torch.stack([data[i+1:i+block_size+1] for i in ix]) return x.to(device), y.to(device) @torch.no_grad() def estimate_loss(): out = {} model.eval() for split in ['train', 'val']: losses = torch.zeros(eval_iters) for k in range(eval_iters): X, Y = get_batch(split) _, loss = model(X, Y) losses[k] = loss.item() out[split] = losses.mean() model.train() return out # ========== Bigram 语言模型 ========== class BigramLanguageModel(nn.Module): def __init__(self, vocab_size): super().__init__() # 整个模型只有一个 Embedding 层 [vocab_size, vocab_size] self.token_embedding_table = nn.Embedding(vocab_size, vocab_size) def forward(self, idx, targets=None): logits = self.token_embedding_table(idx) # [B, T, C] if targets is None: return logits, None B, T, C = logits.shape logits = logits.view(B*T, C) # [B*T, C] targets = targets.view(B*T) # [B*T] loss = F.cross_entropy(logits, targets) return logits, loss def generate(self, idx, max_new_tokens): for _ in range(max_new_tokens): logits, _ = self(idx) # [B, T, C] logits = logits[:, -1, :] # [B, C] 只取最后位置 probs = F.softmax(logits, dim=-1) # [B, C] idx_next = torch.multinomial(probs, num_samples=1) # [B, 1] idx = torch.cat((idx, idx_next), dim=1) # [B, T+1] return idx # ========== 训练 ========== model = BigramLanguageModel(vocab_size).to(device) optimizer = torch.optim.AdamW(model.parameters(), lr=learning_rate) for iter in range(max_iters): if iter % eval_interval == 0: losses = estimate_loss() print(f"step {iter}: train loss {losses['train']:.4f}, val loss {losses['val']:.4f}") xb, yb = get_batch('train') logits, loss = model(xb, yb) optimizer.zero_grad(set_to_none=True) loss.backward() optimizer.step() # ========== 生成文本 ========== context = torch.zeros((1, 1), dtype=torch.long, device=device) print(decode(model.generate(context, max_new_tokens=500)[0].tolist()))
vs Transformer

Bigram vs Transformer 架构对比

Bigram 模型

模型层数: 仅 1 层（Embedding）
注意力机制: 无，每 token 只看当前
位置编码: 无，不感知位置
上下文: 只看当前 token，不看上下文
FFN: 无
参数量: 65 × 65 = 4,225
预测方式: 每个位置直接输出下一个 token 的 logit

Transformer 模型

模型层数: 6 层 Encoder + 6 层 Decoder
注意力机制: Multi-Head Self-Attention + Cross-Attention
位置编码: Sin/Cos 位置编码，注入序列顺序
上下文: 每个位置可关注序列中所有其他位置
FFN: 512 → 2048 → 512，每层都有
参数量: 约 60M
预测方式: Decoder 自回归，每步预测下一个 token
