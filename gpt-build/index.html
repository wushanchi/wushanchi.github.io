01Self-Attention 机制

核心公式

Attention(Q, K, V) = softmax(QK^T / √d_k) V

直观理解

Q: "我想要查询什么？"
K: "我包含什么信息？"
V: "我的实际内容是什么？"

相似度 = Q · K^T
归一化 = softmax(相似度 / √d)
输出 = Σ(归一化权重 × V)

02Multi-Head Attention

原理

多头数 = n_head
每头维度 = d_k = d_model / n_head

head_i = Attention(Q_i, K_i, V_i)
output = concat(head_1, ..., head_n) * W_o

维度变化

输入
(B, T, d_model)
→
QKV投影
(B, T, 3d_model)
→
分割
(B, n_head, T, d_k)
→
输出
(B, T, d_model)

03Transformer 组件

LayerNorm

LayerNorm(x) = gamma * (x - mean) / sqrt(var + eps) + beta

残差连接

x = x + sublayer(x)

位置编码

PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
